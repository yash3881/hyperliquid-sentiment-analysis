#  Hyperliquid Trader Behavior vs Market Sentiment Analysis

##  Objective
This project analyzes how market sentiment (Fear vs Greed) impacts trader behavior and performance on Hyperliquid.  
The goal is to uncover patterns that can help in building smarter trading strategies.

---

##  Datasets Used

1. Bitcoin Market Sentiment (Fear/Greed)**
   - Columns: Date, Classification
   - Provides daily sentiment label (Fear / Greed)

2. Historical Trader Data (Hyperliquid)**
   - Includes: account, symbol, execution price, size, side, time, leverage, closedPnL, etc.
